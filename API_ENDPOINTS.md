# API Endpoint List

## Books

- `GET /api/books`
- `GET /api/books/{id}`
- `POST /api/books`
- `PUT /api/books/{id}`
- `DELETE /api/books/{id}`

## Members

- `GET /api/members`
- `GET /api/members/{id}`
- `POST /api/members`
- `PUT /api/members/{id}`
- `DELETE /api/members/{id}`

## Borrow Records

- `GET /api/borrow-records`
- `GET /api/borrow-records/member/{memberId}`
- `POST /api/borrow-records/borrow`
- `POST /api/borrow-records/return`
