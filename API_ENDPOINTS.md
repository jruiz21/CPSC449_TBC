# API Endpoint List

## Books

- `GET /api/books`
- `GET /api/books/{id}`
- `POST /api/books`
- `PUT /api/books/{id}`
- `DELETE /api/books/{id}`

## Members

- `GET /api/members`
- `GET /api/members/{id}`
- `POST /api/members`
- `PUT /api/members/{id}`
- `DELETE /api/members/{id}`

## Borrows

- `GET /api/borrows`
- `GET /api/borrows/member/{memberId}`
- `POST /api/borrows`
- `POST /api/borrows/return`
